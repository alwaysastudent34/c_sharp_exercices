📝 Razor Pages Web App with ASP.NET Core
Created on: February 2025

📖 About the Project | À propos du projet
🇬🇧 English
This project is a Razor Pages web application built using ASP.NET Core. The tutorial series follows a step-by-step guide to develop a dynamic web application that integrates database functionalities using Entity Framework Core.

🇫🇷 Français
Ce projet est une application web Razor Pages développée avec ASP.NET Core. La série de tutoriels suit un guide pas à pas pour développer une application web dynamique intégrant des fonctionnalités de base de données avec Entity Framework Core.

🎯 Features | Fonctionnalités
✅ Create, Read, Update, Delete (CRUD) operations
✅ Database integration with SQL Server (LocalDB)
✅ Scaffolded Razor Pages for automated UI generation
✅ Search functionality for filtering data
✅ Form validation using Data Annotations
✅ Deployment-ready structure

🛠️ Technologies Used | Technologies utilisées
ASP.NET Core (Razor Pages)
Entity Framework Core
SQL Server (LocalDB)
C#
HTML, CSS

🔎 Usage | Utilisation
1️⃣ Open the Movies page to view the movie list.
2️⃣ Click Create New to add a new movie.
3️⃣ Use the Search bar to filter movies by title.
4️⃣ Click Edit to modify a movie's details.
5️⃣ Click Delete to remove a movie.
